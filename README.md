# Church & School Websites - Portfolio Project

This repository contains two beautiful, modern websites built with Next.js, TypeScript, Tailwind CSS, and Framer Motion - perfect for your portfolio!

## Projects

### 1. Grace Church Kenya Website (`church-website/`)
A stunning church website featuring:
- Floating transparent navbar
- Beautiful purple-pink gradient theme
- Animated hero section with service information
- Sections for About, Services, Events, Gallery, and Contact
- Smooth scroll behavior and animations
- Fully responsive design

**[View Church Website README](./church-website/README.md)**

### 2. Excellence Academy Kenya Website (`school-website/`)
A professional school website featuring:
- Dynamic floating navbar
- Blue-teal gradient color scheme
- Comprehensive program information
- Clear admissions process
- Interactive gallery and contact form
- Mobile-first responsive design

**[View School Website README](./school-website/README.md)**

## Tech Stack

Both projects are built with:
- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS 4** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Beautiful icon library
- **Geist Font** - Modern font family

## Quick Start

### Running Church Website
```bash
cd church-website
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### Running School Website
```bash
cd school-website
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

## Customization Guide

Both websites are highly customizable:

1. **Colors**: Update Tailwind color classes in components
2. **Content**: Modify text in component files
3. **Branding**: Change logos and names in Navbar components
4. **Metadata**: Update SEO information in `app/layout.tsx`
5. **Contact Info**: Update details in Contact components

## Deployment

Both sites can be easily deployed to:
- **Vercel** (Recommended)
- **Netlify**
- **AWS Amplify**
- Any platform supporting Next.js

### Deploy to Vercel
```bash
# From each project directory
npm run build
vercel deploy
```
